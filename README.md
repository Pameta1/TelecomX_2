# 📊 Predicción de Cancelación de Clientes (Customer Churn)

## 📌 Descripción del Proyecto

Este proyecto tiene como objetivo analizar y predecir la cancelación de clientes en una empresa de telecomunicaciones utilizando técnicas de análisis de datos y machine learning.

---

## 🧹 Limpieza de Datos

Durante el análisis se detectaron valores faltantes en la variable **TotalCharges**.
